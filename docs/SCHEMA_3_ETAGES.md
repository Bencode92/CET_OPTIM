# Schéma d'optimisation CET → PERCOL en 3 étages

## Vue d'ensemble

```
┌─────────────────────────────────────────────────────────────┐
│ ÉTAGE 1 — ENTRÉE (mois N)                                   │
│ Intérimaire met X% IFM/ICCP en CET                         │
│ → Potentiel gain RGDU (petit levier)                        │
│ → Cash disponible pour l'ETT                                │
├─────────────────────────────────────────────────────────────┤
│ ÉTAGE 2 — DÉTENTION (mois N à N+12)                         │
│ ETT détient le cash, sert rendement CET (3-5%)              │
│ ETT peut placer la trésorerie (4-5%)                        │
│ → Spread de trésorerie (modeste)                            │
├─────────────────────────────────────────────────────────────┤
│ ÉTAGE 3 — SORTIE (mois N+12)                                │
│                                                             │
│  Option A: Cash         │  Option B: PERCOL                 │
│  Cotis. patronales ~48% │  Forfait social 16-20%            │
│  Gain RGDU annulé       │  Gain RGDU conservé               │
│  → Décalage tréso only  │  → ÉCONOMIE 28-32 POINTS ← LEVIER│
└─────────────────────────────────────────────────────────────┘
```

## Avantages intérimaire

- Rendement CET garanti (3-5%, > livret A)
- Exonération IR sur transfert CET → PERCOL
- Rendement PERCOL en plus
- Déblocage anticipé : résidence principale, surendettement, etc.

## Risques

1. **Adhésion** : intérimaires = besoin de liquidité
2. **Plafond 10 jours** : applicable aux sommes monétaires ? À VALIDER
3. **Risque URSSAF** : CET doit avoir finalité sociale légitime
4. **Départ anticipé** : portabilité CET si départ < 12 mois
5. **Forfait social** : 16% vs 20% selon config PERCOL
